
Extensions/Git.Mv.UGit.Extension.ps1
------------------------------------
### Synopsis
Git Move Extension

---
### Description

Outputs git mv as objects.

---
### Examples
#### EXAMPLE 1
```PowerShell
git mv .\OldName.txt .\NewName.txt
```

#### EXAMPLE 2
```PowerShell
git mv .\OldName.txt .\NewName.txt --verbose
```
