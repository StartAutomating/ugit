Extensions/Git.Worktree.ugit.extension.ps1
------------------------------------------

### Synopsis
git worktree extension

---

### Description

Extends git worktree to return objects.

Currently, only `git worktree list` is supported.

---

### Related Links
* [https://git-scm.com/docs/git-worktree](https://git-scm.com/docs/git-worktree)

---

### Examples
> EXAMPLE 1

```PowerShell
git worktree list
```
